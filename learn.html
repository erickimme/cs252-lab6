HOW TO PLAY THE GAME CORRECTLY

"START VIDEO"

ONE.
PLAYER ONE STARTS ON BLUE, PLAYER TWO STARTS ON RED.

TWO.
CURRENT PLAYER SELECTS A WORD.

THREE.
WORD MUST START FROM A BLOCK HE/SHE ALREADY OWNS.

FOUR.
CURRENT PLAYER CAN CLEAR WORD TO START OVER.

FIVE.
CURRENT PLAYER CAN PRESS SELECTED TO UNSELECT.

SIX.
CURRENT PLAYER CAN CUT OFF OPPONENT'S PATH.

SEVEN.
CURRENT PLAYER CAN CHECK WORD TO GET DEFINITION.

EIGHT.
IF VALID WORD, SWITCH TO NEXT PLAYER.

NINE.
FIRST TO OTHER PLAYER'S BASE WINS.
